# Tech Challenge FIAP - Fase 04

Projeto desenvolvido para o Tech Challenge da FIAP, com o objetivo de criar um modelo preditivo capaz de classificar o nível de obesidade de uma pessoa com base em características físicas, hábitos alimentares e estilo de vida.

## Arquivos do projeto

- `Obesity.csv`: base original utilizada no projeto.
- `modelo_obesidade.pkl`: modelo de Machine Learning treinado.
- `obesity_tratado.csv`: base tratada para análise e dashboard.
- `app.py`: aplicação preditiva desenvolvida em Streamlit.
- `requirements.txt`: bibliotecas necessárias para execução do projeto.

## Modelo utilizado

O modelo final escolhido foi o Random Forest, com acurácia aproximada de 98% na base de teste.

## Objetivo

Auxiliar a equipe médica na identificação do nível de obesidade de pacientes, utilizando Machine Learning como apoio à tomada de decisão.
